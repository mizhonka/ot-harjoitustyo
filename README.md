## Asennus

1. Pura zip-tiedosto ja navigoi komentorivillä puretun hakemisto sisällä "miinaharava"-hakemistoon.

2. Asenna riippuvuudet komennolla
```
poetry install
```

3. Käynnistä sovellus komennolla
```
poetry run invoke start
```
